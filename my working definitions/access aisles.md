# What is a crossing island:

A access aisle is a marked area inside of a parking lot that allows pedestrians to go from other footways like sidewalks into the parking lot to access their cars, these are usually adjacent to disabled parking to make it easier for people with disabilities to get onto the dedicated pedestrian infra and are required in the US by law. These are usually differentiated solely by markings (in Fl these are usually `ladder:skewed`), these markings are encoded using `access_aisle:markings=*`. The identifier tag used to tag that a way represents a access aisle is `footway=access_aisle`. As these are usually painted onto the parking lot they will usually have the same `surface=*` tag as the parking lot and adjacent parking spaces.
